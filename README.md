    </li>
    <li>Navigate to the SurfaceMap directory:
        <pre><code>cd SurfaceMap</code></pre>
    </li>
    <li>Install the required dependencies:
        <pre><code>pip install -r requirements.txt</code></pre>
    </li>
    <li>Run the SurfaceMap tool:
